Frequently Asked Questions

Wo werden die in DOTS erhobenen Daten gespeichert?

Der DOTS Server ist ein Amazon-EC2-Cloudserver, der in Irland steht, d.h. die Daten verlassen die EU nicht. Es ist aber anzumerken, dass der Server-Administrator prinzipiell alle gespeicherten Daten einsehen kann. Bei entsprechender Anonymisierung, die in aller Regel sowieso erforderlich ist, sollte dies aber in den wenigsten Fällen ein datenschutzrechtliches Problem darstellen.

Wie werden die Daten gespeichert und wie komme ich daran?

Die Daten können entweder im CSV-Format oder RDS-Format heruntergeladen werden. Alle Testergebnisse werden als Rohdaten und als aggregierte Scores (z.B. Mittelwerte, Summenscores, Item Response Ability und Standard Error of Measurement) geliefert. Wo sinnvoll, werden Subskalen der Messinstrumente separat berechnet und gespeichert. Der Zugriff erfolgt über die Admin-Konsole, für die Sie ein individuelles Passwort bei der Einrichtung Ihres Zugangs festlegen können.

Kann ich die Testbatterie in einen Onlinesurvey bei einem Drittanbieten integrieren?

Dies geht nur indirekt, z. B. über einen Link von der Umfrage zur Testbatterie (oder umgekehrt). In der Regel kann dort eine CASE-ID (für die jeweilige Versuchsperson) im Link an die Batterie übergeben werden, so dass das Zusammenführen der beiden Datensätze über diese ID in der Regel ohne Probleme möglich ist.
