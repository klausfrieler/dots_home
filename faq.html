Frequently Asked Questions

Wo werden die Daten gespeichert?

Der DOTS Server ist ein Amazon EC2 Cloudserver in der Domäne Irland, d.h. die Daten verlassen die EU nicht. Es ist aber anzumekren, dass der Server-Administrator prinzipiell die gespeicherten Daten einsehen kann.

Wie werden die Daten gespeichert und wie komme ich daran?

Die Daten können entweder im CSV-Format oder RDS-Format abgeholt werden. Alle Testergebnisse werden als Rohdaten und als ausgewertete Scores geliefert, ggf. für die einzelnen Subskalen der Messinstrumente getrennt. Zugriff erfolgt über die Admin-Konsole für die Sie ein individuelles Passwort bei der Einrichtung festlegen können.

Kann ich die Testbatterie in andere Onlinesurveys von Drittanbieten integrieren?

Die geht nur indirekt, z.B. über einen Link von der Umfrage zur Testbatterie (oder umgekehrt). In der Regel kann dort eine CASE-ID (o.ä.) von dor im Link an die Batterie übergeben werden, so dass eine Kombination der beiden Datensätze ohne Probleme möglich ist.
