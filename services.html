Der Service

Die Hauptaufgabe von DOTS ist das Hosting von individuell zusamengestellten Testbatterien auf einem öffentlich zugänglichen und leistungsstarken Server. Ein weiteres Angebot ist die Implementierung neuer Tests mit psychTestR, die dann über DOTS verwendet werden können. Dies kann auch die Re-Implementierung historischer Tests (z.B. die Musikalitätstests von Seashore, Wing und Bentley) zu Lehrzwecken einschließen.

Einrichtung einer Testbatterie

Kostenfrei

Um eine Testbatterie auf DOTS laufen zu lassen, genügt es, sich eine Batterie aus den verfügbaren Tests zusammenzustellen und die Wünsche dem Einrichter (zur Zeit Klaus Frieler) mittzuteilen. Dieser wird in Absprache die Testbatterie aufsetzen und freischalten. Die Laufzeit kann individuell vereinbart werden. Dieser Service ist kostenfrei für DGM-Mitglieder, allerdings behält sich die DGM vor, bei sehr großen Teilnehmerzahlen einen anteiligen Kostenbeitrag festzulegen. Nicht-Mitglieder sprechen uns bitte an, sicherlich lässt sich eine Lösung finden. Die Details können jeweils individuell vereinbart werden.

Oft ist eine Integration einer psychTestR-Testbatterie mit individuellen Komponenten (Test oder Fragebögen) gewünscht. Wenn diese Komponenten nicht von allgemeinen Interesse sind, lohnt sich in der Regel nicht, eine eigene psychTestR zu implementieren. In diesem Fall kann eine Batterie über spezielle Links auch mit einem Fragebogen oder einem Test auf einer anderen Plattform verknüpft werden. Auch dabei beraten wir Sie gern.

Falls Sie also DGM-Mitglied sind und eine Testbatterie laufen lassen wollen, wenden Sie einfach per Mail an Klaus Frieler .

Implementierung neuer Tests und Fragebögen

Kosten nach Vereinbarung

Wenn Sie einen eigenen Hörtest oder Fragebogen entwickeln, kann es für Sie interessant sein, eine psychTestR-Implementation zu erstellen. Diese Implementierung können Sie selbst vornehmen oder in Zusammenarbeit mit den DOTS-Entwicklern umsetzen. Die Implementieurung eines neuen Testinstruments in psychTestR hat einige Vorteile

  • Offizielle Referenzimplementation, z.B. als Supplement für ein Paper.
  • Freie Verfügbarkeit Ihres Testinstruments für andere, interessierte Forscher:innen
  • Unterstützung von Open Science und transparenten Testingmethoden.
  • Größere Sicherheit, dass Ihr Testinstrument von anderen korrekt angewendet werden.
  • Bessere Verbreitung des eigenen Testinstrumentes.

Ansprechpartner für die Implementierung neuer Testinstrumente ist Klaus Frieler. Aufgrund des hohen Aufwands einer solchen Implementation, kann dieser Service leider nicht kostenfrei angeboten werden. Kosten können im Vorfeld veranschlagt werden und so ggf. im Budget von größeren Projekten berücksichtigt werden.

Hosting von Shiny-Apps

Kostenfrei, App-Entwicklung nach Vereinbarung

Da es sich um einen Shiny-Server handelt, können nicht nur Tests gehostet werden, sondern auch andere Formen von Shiny-Apps, wie zum Beispiel zur Konfiguration von komplexen Tests (z.B. der HALT Configurator, oder der Gold-MSI Configurator) oder auch zur Datenauswertung und -visualisierung. Falls Sie eine Shiny-App hosten lassen wollen oder Hilfe bei der Entwicklung gewünscht ist, sprechen Sie Klaus Frieler an.
